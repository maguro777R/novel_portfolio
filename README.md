# novel_portfolio
小説のポートフォリオサイトを作ります

## Preview

```bash
npm install
npm run dev
```

開発サーバー起動後、表示されたローカルURLで川野マグロの小説ポートフォリオサイトを確認できます。

## Files

- `src/`: React + Tailwind のサイト本体
- `src/data/site.ts`: 作品、実績、ノート、お知らせ、プロフィール、問い合わせ情報
- `src/assets/generated/`: 実装で使用する生成画像素材
- `assets/ui-mockup-*.png`: 参照用UIモック画像
- `assets/editorial-still-life.png`: 初期ヒーロー用ビジュアル素材
